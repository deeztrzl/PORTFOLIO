# Audio Assets

This directory contains audio files for the portfolio.

## Files

### arcade-bgm.mp3
Background music for the arcade portfolio. 

**How to add:**
1. Generate using `audio-generator.html` in the root
2. Or find free retro music from:
   - OpenGameArt.org
   - FreePD.com
   - YouTube Audio Library
3. Convert to MP3 format if needed
4. Place file here as `arcade-bgm.mp3`

## Format Requirements

- **Format**: MP3 (H.264 audio)
- **Bitrate**: 128kbps recommended
- **Sample Rate**: 44.1kHz
- **Channels**: Mono or Stereo
- **Max Size**: 2MB recommended
